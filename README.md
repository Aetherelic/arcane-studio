# ✦ Arcane Studio ✦

### Rice harder. Break less.

Arcane Studio is a collection of tools for building beautiful, powerful, and safer Linux desktops.

It is made for people who love ricing, Hyprland, dotfiles, aesthetic workflows, and deep system customization — without blindly trusting every random install script they find online.

## Projects

### ✦ Arcane Guard

Arcane Guard is the safety inspection module.

It scans AUR packages, PKGBUILDs, install scripts, setup scripts, and dotfiles for suspicious or review-worthy patterns before you run them.

Repository:

https://github.com/Aetherelic/arcane-guard

Status:

0.1 release candidate

---

### ✦ Arcane Themes

Arcane Themes is the theme and rice-management module.

It is planned to handle theme snapshots, restoration, wallpaper-driven colors, Hyprland configs, Quickshell layouts, terminal themes, and complete desktop aesthetic profiles.

Repository:

Coming soon

Status:

## Vision

Arcane Studio exists for one reason:

Make Linux customization feel powerful, beautiful, and safer.

Linux ricing should be expressive and fun, but it should not require blindly running unknown scripts with no idea what they do.

Arcane Studio aims to become a full customization suite for people who want control over their desktop, their themes, and their system.

## Philosophy

Arcane Studio is built around three ideas:

- aesthetic desktops should still be practical
- customization should be reversible
- users should understand what they are running
